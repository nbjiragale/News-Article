# Icons To Add

Paste downloaded icon files in this folder first.

Recommended format:
- SVG, 24px, single color, rounded style if available.
- Use simple lowercase filenames with underscores.

Needed icons:
- `ic_arrow_back.svg` - back button in reader and recents.
- `ic_text_format.svg` - reading text settings button.
- `ic_bookmark_border.svg` - bookmark action in top bar.
- `ic_history.svg` - recents action.
- `ic_volume_up.svg` - pronunciation/audio button in meaning sheet.
- `ic_menu_book.svg` - example-in-context section.
- `ic_bookmark_add.svg` - save word action.
- `ic_school.svg` - practice action.
- `ic_article.svg` - article/empty recents state.
- `ic_close.svg` - dismiss sheet or close actions if needed later.

Optional icons:
- `ic_search.svg` - search/filter recents later.
- `ic_delete.svg` - delete a recent article later.
- `ic_translate.svg` - language/meaning feature label.

Note:
Android cannot use nested folders inside `res/drawable`. After you paste icons here, we can convert SVGs to vector drawables and place them in `app/src/main/res/drawable` as `ic_*.xml`.
